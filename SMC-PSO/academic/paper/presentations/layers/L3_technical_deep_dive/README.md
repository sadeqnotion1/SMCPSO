# L3 Technical Deep Dive

Use this level for engineering and implementation review sessions.

## Audience

- technical reviewers
- research engineers
- collaborators working on the codebase

## Target Length

- 40 to 80 slides
- 60 to 120 minutes

## Focus

- detailed architecture
- algorithms and parameter choices
- implementation details
- validation results and failure analysis

## Material Checklist

- `slides/`: technical deep-dive deck
- `speaker_notes/`: implementation-level explanations
- `figures/`: architecture diagrams and performance plots
- `code_snippets/`: representative implementation excerpts
- `handouts/`: technical summary appendix
- `references/`: key technical sources
