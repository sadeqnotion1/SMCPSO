# L2 Foundations

Use this level for teaching core concepts and project basics.

## Audience

- students
- new lab members
- onboarding engineers

## Target Length

- 20 to 40 slides
- 30 to 60 minutes

## Focus

- domain background
- core model and assumptions
- method intuition
- simple examples and demos

## Material Checklist

- `slides/`: concept-first deck with definitions
- `speaker_notes/`: explanatory scripts and examples
- `figures/`: foundational diagrams and flow charts
- `code_snippets/`: short illustrative snippets
- `references/`: introductory reading list
